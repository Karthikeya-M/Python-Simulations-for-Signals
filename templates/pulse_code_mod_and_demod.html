About Pulse Code Modulation (PCM) and Demodulation

Pulse Code Modulation (PCM) is a digital representation of an analog signal. The process involves sampling the analog signal at discrete time intervals, quantizing the sampled values into a finite set of levels, and encoding these values into binary form.

This page allows you to experiment with PCM by selecting parameters such as sampling frequency, message frequency, amplitude, number of bits per sample, and signal duration. The demodulation process reconstructs the original signal from the quantized samples.

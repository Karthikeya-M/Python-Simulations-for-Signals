This application allows you to apply different image processing operations using OpenCV. Follow the steps below to use the tool:

  • Upload an image from your device
  • Select an operation from the dropdown menu
  • If the selected operation requires parameters (like Resize or Rotate), enter the values
  • Click Process to see the result
  • Processed image will be displayed on the right side
